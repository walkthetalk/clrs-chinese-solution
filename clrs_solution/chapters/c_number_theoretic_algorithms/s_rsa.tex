%e31.7-1
\startEXERCISE
證明：在如下意義中， RSA 是乘法的：
\startformula
P_A(M_1)P_A(M_2) \equiv P_A(M_1 M_2) (\mod n)
\stopformula
利用這個事實證明：
如果對方有一個過程，
對 \m{\integers_n} 中的用 \m{P_A} 加密的消息，
他能夠有效地解密出其中的百分之一，
則他可以運用一種概率性算法，
以較大概率爲每一條用 \m{P_A} 加密的信息進行解密。
